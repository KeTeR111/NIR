ИНСТРУКЦИЯ ПО ЗАПУСКУ ГИДРОДИНАМИЧЕСКОГО ДАШБОРДА

1. ПРОСТОЙ СПОСОБ (Windows):
   - Дважды щелкните по файлу "install_and_run.bat"
   - Следуйте инструкциям на экране
   - После установки дашборд запустится автоматически
   - Откройте браузер и перейдите по адресу: http://127.0.0.1:8050

2. ЕСЛИ УЖЕ УСТАНОВЛЕН PYTHON:
   - Дважды щелкните по файлу "run.bat" (Windows)
   - Или запустите в терминале: python dashboard.py

3. РУЧНАЯ УСТАНОВКА:
   - Установите Python с сайта python.org
   - Откройте командную строку/терминал в папке с дашбордом
   - Выполните команды:
        pip install -r requirments_to_dashboard.txt
        python dashboard.py

4. ДЛЯ MAC/LINUX:
   - Запустите файл "install_and_run.sh" в терминале:
        chmod +x install_and_run.sh
        ./install_and_run.sh

ПРОБЛЕМЫ И РЕШЕНИЯ:

- Если возникает ошибка "Python не найден":
  Скачайте и установите Python с python.org

- Если дашборд не открывается в браузере:
  Проверьте что адрес правильный: http://127.0.0.1:8050

- Если порт 8050 занят:
  Дашборд автоматически попробует использовать другой порт

ПАПКА С ДАННЫМИ:
Убедитесь, что папка "Results" с данными находится в той же папке, что и дашборд.
